# WEBTECHNOLOGY-Project

## Local development (Vue 3 + Vite)

This repository now contains a minimal Vue 3 + Vite single-page application with `vue-router`, example views, an async fetch demo, and a contact form with client-side validation.

Run locally:

```bash
npm install
npm run dev
```

Open the address shown in the terminal (usually `http://localhost:5173`).
